# Administrator's Guide

The Administrator's Guide covers the management and
configuration tasks for the pgEdge AI DBA Workbench.
Administrators control user access, database connections,
alert rules, notification channels, probes, and API
access.

## Topics

The Administrator's Guide includes the following topics:

- [Authentication](authentication.md) covers user
  accounts, service accounts, API tokens, and
  role-based access control.
- [Connections](connections.md) describes how to manage
  monitored database connections.
- [Alert Rules](alert-rules.md) explains how to
  configure alert thresholds and hierarchical overrides.
- [Notification Channels](notification-channels.md)
  covers email, Slack, Mattermost, and webhook
  notification delivery.
- [Probes](probes.md) describes how to enable, disable,
  and configure data collection probes.
- [API Reference](api/reference.md) provides interactive
  documentation for the REST API.
- [Server Information](api/server-info.md) documents
  the server information and AI analysis endpoints.
- [Metrics](api/metrics.md) describes the metrics
  query tools and available probes.
