# Web Client Documentation

The pgEdge AI DBA Workbench web client provides a browser-based
interface for monitoring and managing PostgreSQL database estates.
The client connects to the MCP server and displays real-time
metrics, alerts, and AI-generated insights.

## Overview

The web client includes the following features:

- Hierarchical monitoring dashboards display metrics at the
  estate, cluster, server, database, and object levels.
- The cluster navigator provides tree-based navigation across
  groups, clusters, and individual servers.
- An alert panel displays active alerts with grouped views
  and LLM-powered analysis.
- The event timeline shows configuration changes, restarts,
  and other notable events across selected servers.
- An AI overview panel presents LLM-generated summaries of
  database health and status.
- The admin panel manages users, groups, tokens, probes,
  alert rules, and notification channels.
